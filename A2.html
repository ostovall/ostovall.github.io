Assignment 2 – Arduino LED Circuit and Code

Overview

This project demonstrates a basic Arduino circuit that uses three LEDs and a button input to create interactive lighting patterns using digital and analog functions. The program cycles two LEDs with digital control and fades a third LED using PWM. The circuit uses resistors to limit LED current and a pull-down resistor for the button input.

Schematic and Circuit

Components:

  • Arduino Uno
  • 3 LEDs (Red, Yellow, Green)
  • 3 × 220Ω resistors (LED current limiters)
  • 1 × 10kΩ resistor (button pull-down)
  • 1 Pushbutton
  • Breadboard and jumper wires

Resistor Calculation

For each LED:
R = (5V − 2V) / 0.015A = 200Ω → use 220Ω (standard value)

For the button:
A 10kΩ pull-down resistor ensures the pin reads LOW when not pressed, drawing only 0.5mA of current.

Code


// --- Define pins ---
const int buttonPin = 2;                 // the button input pin
const int ledPins[3] = {9, 10, 11};      // LED output pins (PWM pins support analogWrite)

int buttonState = LOW;

void setup() {
  pinMode(buttonPin, INPUT);
  for (int i = 0; i < 3; i++) {
    pinMode(ledPins[i], OUTPUT);
  }
}

void loop() {
  buttonState = digitalRead(buttonPin);

  if (buttonState == HIGH) {
    for (int i = 0; i < 2; i++) {
      digitalWrite(ledPins[i], HIGH);
      delay(150);
      digitalWrite(ledPins[i], LOW);
      delay(150);
    }

    for (int brightness = 0; brightness <= 255; brightness += 5) {
      analogWrite(ledPins[2], brightness);
      delay(15);
    }
    for (int brightness = 255; brightness >= 0; brightness -= 5) {
      analogWrite(ledPins[2], brightness);
      delay(15);
    }

  } else {
    for (int i = 0; i < 3; i++) {
      digitalWrite(ledPins[i], LOW);
    }
  }
}

Battery Life Calculation

Average current draw ≈ 35mA (15mA per LED + 20mA Arduino idle).
1200 mAh / 35 mA ≈ 34 hours runtime.

Worst case (all LEDs on): 1200 mAh / 65 mA ≈ 18 hours.

Measured LED Voltage

Measured forward voltage (red LED): 1.95V
Theoretical forward voltage: ~2.0V
Difference is within typical tolerance range.
